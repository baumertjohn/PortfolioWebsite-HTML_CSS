Brewery Finder Website

A custom API based website

This project running on a Flask server that displays results from the Open Brewery DB based on whether a user wants to search by "Brewery Name", "City" or "State". The results are displayed with basic brewery information and a map provided by Google Maps. Since multiple pages of results can be returned, a solution was needed to set up multiple pages.

Image Color Finder

A website that finds the most common colors in an uploaded image.

A basic website running on a Flask server that utilizes Python and Pandas to determine the top 10 colors of a user uploaded image and then returns the top 10 colors.

Simple To-Do List

This project utilizes Python and Flask to create a Todo List styled website that allows a user to login and save a list in the database.

There is a "test" account in the database with the following credentials:
Email: test@email.com
Password: 1234
After logging in / creating an account you can now "add" a task which is auto time stamped. A task can be "checked off" by clicking the square proceeding the appropriate task which moves it to "Done". From here the task can be "deleted" by clicking the "X".
